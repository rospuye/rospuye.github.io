Automatic Commentator For Robotic Football


ABOUT THE PROJECT

The goal of this project is the development of an automatic system for natural language commentary of robotic (or human) football games. Using this low level information (such as player and ball positions), which is available through Simulation League and can be sourced from footage by the Medium-Sized Robot League or from human football footage, the system must generate higher level information, recognizing ball passes, penalty kicks, play opportunities, counter-attacks and other football events, as well as analyse the general state of the game by estimating ball possession, team formations, preferencial attack/defence zones, success rates, territorial domain, among others. This information must be used to generate an interesting and appealing commentary in real time and using natural language. The commentary to be generated might eventually be parameterizable, for example, in terms of emotion or commentary partiality.
